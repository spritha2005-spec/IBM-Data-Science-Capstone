# IBM Applied Data Science Capstone

This repository contains the complete IBM Applied Data Science Capstone project focused on SpaceX Falcon 9 launch data analysis and machine learning prediction.

## Project Stages
- Data Collection using API
- Web Scraping
- Data Wrangling
- Exploratory Data Analysis
- SQL Analysis
- Interactive Visual Analytics
- Machine Learning Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium
- Scikit-learn
- SQL
- Jupyter Notebook

## Objective
Analyze SpaceX launch data and build machine learning models to predict the success of Falcon 9 first-stage landings.
